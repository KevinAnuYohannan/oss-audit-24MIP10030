## Project Description
This project is an Open Source Software Audit based on VLC Media Player. It covers the origin, licensing, ethical aspects, Linux footprint, and ecosystem of the software. It also includes a comparison between open-source and proprietary software.

## Scripts Included

1. Script 1 - System Identity Report  
   Displays system information such as user, kernel version, and uptime.

2. Script 2 - VLC Package Inspector  
   Checks whether VLC is installed and shows its details.

3. Script 3 - Disk and Permission Auditor  
   Displays directory permissions and disk usage.

4. Script 4 - Log File Analyzer  
   Counts occurrences of keywords in a log file.

5. Script 5 - Open Source Manifesto Generator  
   Generates a personalized open-source statement.

## How to Run Scripts
chmod +x script1.sh
./script1.sh

Repeat for other scripts.
